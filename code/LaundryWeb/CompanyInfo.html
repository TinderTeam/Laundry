快客洗涤公司简介

News Start

        十九世纪中叶，在法国巴黎一个普通的服装店里，年轻的裁缝乔利·贝朗正在为一件不小心被煤油污染的 贵重衣服而焦急万分：这怎么赔得起啊?但煤油浸过的地方不仅没有污痕，反倒比别处显的更加干净。乔利因此 受到启发，1855年，法国巴黎诞生了世界上第一家干洗店。在此后漫长的时间里，做为世界服装之都的一个新 兴行业，干洗业在这个拥有灿烂文明和悠久历史的浪漫国度里，孕育、滋养、发展着，时至今日，干洗业已经 发展成为一个跨行业、多学科、世界性的朝阳行业。

　　 “Drolohq”，作为法国干洗业界的知名品牌，2000年经过特许进入中国，在儒家文化的溯源地山东济南 成立了朵拉国际洗衣连锁中国总部-济南朵拉洗业有限公司，实现了两种文化的完美契合。

“Drolohq”秉承法兰西悠久的洗业文化，整合全球顶极的洗业资源，不断追求诚信、完美、卓越的个性化品牌服务，经过11年的发展，带有金百合标志的“Drolohq”高质洗衣连锁店已成为国内最具实力和影响力的大型洗衣连锁特许加盟机构之一，在国内拥有数十家连锁洗业公司，数百家加盟代理商，拥有近千名洗衣技师，已经为中国几千万消费者提供了绿色、健康、环保的专业洗衣服务。

　　作为国际洗涤协会常务理事单位，“Drolohq”始终坚持把洗涤设备和耗材的研制以及特许经营模式的探索和发展作为品牌发展的助力，通过“Drolohq”独有的高科技、高质量、高标准的“高质洗衣”理念，融合“Drolohq”绿色、健康、环保的洗业文化，全力打造最适合于中国的特许经营模式和“Drolohq”优质、高效的运营支持系统，相信“Drolohq”必将和志同道合的投资同盟在中国市场再次赢得胜利和辉煌，并保持“Drolohq”品牌在中国洗涤行业的领先地位，进一步实现品牌价值的社会化，全力推动中国洗业健康、持续的发展，为中国更多的消费者创造高品质的生活。
