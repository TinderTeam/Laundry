公司简介

    贵州省金州快客清洁服务有限公司是一家集干洗衣物、洗涤技术咨询培训，室内保洁及物业管理等多项服务为一体的O2O模式专业环保型清洁服务公司。公司管理机制完善，清洁作业采用专业清洗设备结合员工丰富的经验来进行工作流程；洗涤技术严格实行上岗前操作培训采用国内知名洗染技术及配方，让洗衣事故尽可能的减少，取得广大顾客的认可，凭着较高的洗涤、清洁质量和严格的管理制度，为众多企事业单位、酒店等提供优质的专业服务，特称之为“快客洗涤”。

    公司概况：贵州省金州快客清洁服务有限公司经过长期的策划和筹备，于2014年12月1日正式成立，注册资金200万元，自有多名掌握较好的干洗技术的技术骨干和多名从事多年家政保洁的员工，经集中进行专业技术及道德的培训，我公司员工自身素质较好。

    企业文化：洗涤满意、服务满意、形象满意、尊重顾客、理解顾客持续提供超越顾客期望的产品的服务与质量；自我完善服务系统，加强洗前上门收取、前后消毒干洗中、洗净后送回的服务，对顾客在洗涤衣物中出现的各种问题及时帮助解决，让顾客们感到极大的方便。

    公司产品：快客洗涤干洗加盟店、洗涤技术咨询与培训、洗涤设备销售、命名为“快客洗涤”的手机APP。
