“快客洗涤”加盟介绍

加盟“快客洗涤”干洗连锁您将享有以下权利：

1、所在地一定区域内的独享经营权，公司承若在同一区域内不再增加“快客洗涤”干洗店面；

2、在指定区域内拥有“快客洗涤”的商标使用权；

3、共享“快客洗涤”的手机APP及微信公众平台；

4、在“快客洗涤”官方网站上展示加盟店并标注地址及服务电话；

5、享有公司提供的运营管理模式使用权；

6、公司给予加盟商提供部分团体会员单位及个人的洗衣业务；

“快客洗涤”为干洗店加盟商开业前期提供以下服务：

1、签订意向性合同后我公司委派从事多年干洗工作人员为您选址提供建议；

2、整个干洗店的门店设计和整体形象设计；

3、由加盟商委派店员到公司总部学习洗染技术；

4、设备运到后派专业洗染人员指导安装及操作；开业后委派技术人员带店指导，建立稳定的客户群；

5、根据门店规模赠送开业礼包及开业策划。

“快客洗涤”为干洗店加盟商运营后期提供以下服务：

1、公司营销部门提供的干洗店促销活动方案；

2、公司提供所在快客洗涤干洗店区域通过手机APP下单的客户；

3、公司每年举办不少于2次集中培训或视频培训，解决店面经营难题，确保加盟商洗涤技术与利润。

整店输出，参照我公司门店装修标准实施，由我公司提供技术培训，按照干洗店标准购置干洗设备器材，使用我公司指定洗涤原料；

1、 收送点：经营面积10-20平米，经营项目为公司总部洗衣收送点，设备金6000-10000元（可选项），包含：消毒柜、成衣包装机、烫台、蒸汽发生器、蒸汽熨斗、包装袋；管理费自第二年收取（按用户量提取），VI形象、门店设计统一按公司标准执行，合约期由加盟商自行决定，无加盟费，无品牌使用费，利润由公司提取分配，及形象保证金5000元（可返）。门店选址由我公司人员提供建议；

2、 标准店：经营面积40-60平米，经营项目为干洗店，利润自收，设备金76000-120000元（可自由选择洗涤设备），包含：干洗机、水洗机、烘干机、衣物自动传输线、衣物消毒柜（单门）、收银软件、成衣包装机、烫台、蒸汽发生器、蒸汽熨斗，管理费自第二年收取（按用户量提取），VI形象、门店设计统一按公司标准执行，免费培训1名员工，开业带店7天，赠送开业礼包，合约期限为三年，加盟费8800元，无品牌使用费，及形象保证金10000元（可返）。门店选址由我公司人员提供建议；

3、 形象店：经营面积60-90平米，经营项目为干洗店，利润自收，设备金139000-239000元，包含：干洗机2台、洗脱机、烘干机、衣物自动传输线、衣物消毒柜（双门）、收银软件带钱箱、成衣包装机、烫台、蒸汽发生器、蒸汽熨斗，管理费自第二年收取（按用户量提取），VI形象、门店设计统一按公司标准执行，免费培训3名员工，开业带店15天，赠送开业礼包，合约期限为五年，加盟费12000元，无品牌使用费，及形象保证金10000元（可返）。门店选址由我公司人员提供建议；

4、 旗舰店：经营面积90平米以上，经营项目为干洗店，利润自收，设备金338000-398000元，包含：干洗机2台、洗脱机、烘干机、衣物自动传输线、衣物消毒柜（双门）、收银软件带钱箱、成衣包装机、烫台2台、蒸汽发生器、蒸汽熨斗，管理费自第二年收取（按用户量提取），VI形象、门店设计统一按公司标准执行，免费培训5名员工，开业带店30天，赠送开业礼包，合约期限为五年，加盟费16000元，无品牌使用费，及形象保证金10000元（可返）。享有“快客洗涤”APP后台店员账户，可直接接收所在地所有顾客下单详情，门店选址由我公司人员提供建议。

以上任一形式的加盟店都必须附带上门收送服务；

除收送点外，任何加盟店面必须使用我公司指定洗涤原料；

所有加盟店面都有维护“快客洗涤”品牌的形象的义务，如接收到没有把握处理的衣物请寄回公司进行处理，寄回公司处理的衣物造成洗涤事故的由公司承担责任。
